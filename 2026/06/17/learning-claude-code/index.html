我学会了用 Claude Code — 一个高考毕业生的编程入门之旅

发表于 2026-06-17 分类于 技术

从零开始，和 AI 一起写代码

高考结束后，我终于有了大把自由时间。除了旅行、看书、补觉之外，我还给自己定了一个小目标——学会做一个属于自己的网站。

说实话，在此之前我对编程的了解几乎为零。HTML、CSS、JavaScript 这些词听过，但从来没动手写过一行代码。

然后我遇到了 Claude Code。

什么是 Claude Code？

简单来说，Claude Code 是一个 AI 编程助手。你可以在终端里用自然语言告诉它你想做什么，它就能帮你写代码、调试、部署。不需要先花三个月学编程基础，直接上手做项目，边做边学。

我第一次打开它的时候，心里想的是：”我就试试看，不行就算了。”

结果——两个小时之后，我的个人主页就上线了。

我做了什么？

这几天，我用 Claude Code 完成了：

  • 🌐 个人主页 — 一个带导航、动画、滚轮效果的响应式单页网站
  • 📝 Hexo 博客 — 从零搭建，配置 NexT 主题，加上液态玻璃设计
  • 🔍 本地搜索 — 弹窗式搜索，支持快捷键，高亮匹配
  • 🎨 毛玻璃 UI — 全局 Dock 级液态玻璃效果，动态渐变背景

每一个功能，我都是直接用中文描述我想要的效果，Claude Code 来帮我实现。比如我说”我想要一个搜索弹窗，按 Ctrl+K 能打开，背景是毛玻璃效果”，它就能直接写出来。

我学到了什么？

虽然大部分代码是 AI 写的，但在这个过程中我不知不觉学会了很多：

  1. 读代码 — 为了确认 AI 写的是不是我想要的，我开始能看懂 HTML 结构和 CSS 样式了
  2. 命令行 — 学会了 git push、hexo deploy 这些基本操作
  3. 调试思维 — 当效果不对时，我学会了描述问题：”汉堡菜单点不开” 比 “网站坏了” 有用 100 倍
  4. 迭代优化 — 先出一个版本，然后一轮一轮改，每一次都比上一次更好

最好的学习方式不是先学完所有理论再动手，而是有一个强大的伙伴陪你一起做项目，遇到问题当场解决。

给同样想入门的朋友

如果你也和我一样是编程小白，我的建议是：

  1. 不要怕 — 你不需要先成为专家才能开始
  2. 描述你想要的效果 — 越具体越好，颜色、动画、交互都说清楚
  3. 多问为什么 — AI 改完之后，问它为什么这样写，你会学得更快
  4. 保持好奇 — 看到一个好看的效果就想着”我能不能也做一个？”

下一步

接下来我想试试：

  • 给博客加上暗色模式切换
  • 写一个评论系统
  • 学一点 React，做个更复杂的应用

无论如何，编程已经从”好难，不想碰”变成了”好有趣，还想做更多”。

感谢 Claude Code 让我这个编程小白也能做出让自己满意的网站。这只是开始。
